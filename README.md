## О себе

Пишу практичные инструменты для сообществ, серверов и рутинных задач. Больше всего люблю проекты, где бот, скрипт или небольшая утилита реально экономят время и закрывают конкретную боль.

- Разрабатываю Telegram-ботов для уведомлений, поддержки, напоминаний и рабочих процессов.
- Делаю Discord-ботов для комьюнити-серверов: музыка, поддержка, магазины, модерация.
- Собираю desktop/CLI-утилиты для администрирования, диагностики и автоматизации.
- Работаю с Minecraft server tools, Paper/Spigot-плагинами и серверными интеграциями.
- Использую Linux, VPS, Docker, базы данных и GitHub в повседневных задачах.
- Изучаю C++ и постепенно расширяю стек в сторону системного и desktop-разработки.
